🐙 GitHub sync

Your data syncs to a private GitHub repository automatically. Follow these steps once to connect.

1
Create a free GitHub account
Go to github.com and sign up if you don't have one yet.
2
Create a private data repository
Click + → New repository, name it malia-finance-data, set it to Private.
3
Create a Personal Access Token
Go to Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token. Tick only repo. Copy the token.
4
Enter your details below
Your token is stored only on this device — it goes directly to GitHub's servers, nowhere else.
