GitHub sync

Your data is stored in a private GitHub repository and synced automatically. Follow these steps once to connect.

1
Create a free GitHub account
Go to github.com and sign up if you don't have one.
2
Create a new private repository
Name it malia-finance-data — make sure it's set to Private.
3
Create a Personal Access Token
Go to Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token. Select only the repo scope. Copy the token.
4
Enter your details below
Your token is stored only on this device — never sent anywhere except directly to GitHub's API.
